# workshop-movie-ticket-booking-calculator
Workshop do Freecodecamp (Certificação em Python), com as praticas como usar booleanos e declarações condicionais em Python, construindo uma calculadora de reserva de ingressos de cinema.

* Workshop.py contém o conteúdo que foi sendo modificado conforme as instruções (de forma gradual, conforme à ordem das solicitações).
* FinalCode.py contém o código final em python do Workshop (de forma rígida, para passar no teste).
* CodigoNovo.py contém o código atual com minhas modificações.
